Static7基础管理系统

开发这个基础系统的初衷:是想帮助解决,后台基础管理系统定制化开发的的问题.
定制化开发讲究对代码的质量要求,代码扩展性和阅读性.有了稳健基础,需求定制化模块或者功能更容易.
使用Layui 2.x前端框架,颜色设计感很棒，清新而不乏深沉，互相柔和，不过分刺激大脑皮层的神经反应，形成越久越耐看的微妙影像。合理搭配，可与各式各样的网站避免违和，从而使你的Web平台看上去更为融洽。
ThinkPHP5是一个很不错的框架. 开发这个基础系统的时候,采用ThinkPHP5.1版本

1． 需要给用户一份Apache Licence ；

2． 如果你修改了代码，需要在被修改的文件中说明；

3． 在延伸的代码中（修改和有源代码衍生的代码中）需要带有原来代码中的协议，商标，专利声明和其他原来作者规定需要包含的说明；

4． 如果再发布的产品中包含一个Notice文件，则在Notice文件中需要带有本协议内容。你可以在Notice中增加自己的许可，但不可以表现为对Apache Licence构成更改。
